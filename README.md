
# 🎓 CGPA & GPA Calculator

A simple and responsive **CGPA & GPA Calculator** built using **HTML, CSS, and JavaScript**. It helps students calculate their **SGPA** and **CGPA** quickly with an easy-to-use interface. The application performs accurate calculations instantly and works seamlessly across desktop and mobile devices.  
**Live Demo:** https://cgpa-gpa-cal.vercel.app/
